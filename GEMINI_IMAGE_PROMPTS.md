## 圖片換入流程（給開發者）

生成圖片後：
1. 存入 `C:\Projects\thunder-blessing-slot\assets\textures\symbols\`
2. Cocos Creator 的 **Assets** 面板會自動顯示新圖片
3. 修改 `ReelManager.ts` 的 `drawCell()` 替換占位符色塊：

```typescript
// 步驟 1：在 GameBootstrap onLoad 載入 SpriteAtlas
// 步驟 2：drawCell 中以 Sprite 元件顯示圖片
const spriteComp = cell.node.getComponentInChildren(Sprite);
spriteComp.spriteFrame = symbolSpriteFrames[sym]; // sym = 'P1', 'P2'...
```

> 不確定如何改？把以下需求貼給 GitHub Copilot：
> "我用 Cocos Creator 3.8，想把 Graphics 色塊替換為 Sprite 顯示圖片，
>  符號代碼對應到 sym_p1.png, sym_p2.png, sym_p3.png, sym_p4.png,
>  sym_l1.png(Z), sym_l2.png(E), sym_l3.png(U), sym_l4.png(S),
>  sym_wild.png, sym_scatter.png。請修改 ReelManager.ts 的 drawCell 函式。"
